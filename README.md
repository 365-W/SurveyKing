
## 什么是卷王

[卷王](http://49.232.153.121:1991/)是一款安装简单、界面美观、易于使用、功能强大的企业级问卷数据采集系统。

## 特性概览

- 安装简单（最快 1 分钟部署）
- 界面美观
- 易于使用（可以快速拖拽创建问卷）
- 响应式布局（所有功能跨终端跨平台自适应）
- 支持多种题型（持续更新）
- 强大的自定义逻辑功能（不输于问卷星、腾讯问卷等）
- 优秀的数据展示编辑功能
- 丰富的报表输出功能（支持多种报表）
- 安全、可靠、高性能的后端 API 服务
- ...
